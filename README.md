# Aventura en el Reino Oscuro

Trabajo final realizado en Python para la materia Paradigmas de Programación.

## Descripción

Es un juego de aventura por consola. El jugador elige una clase, recorre cuatro zonas, combate enemigos, compra mejoras y debe derrotar al jefe final.

## Conceptos utilizados

* Variables y tipos de datos
* Condicionales `if`, `elif` y `else`
* Ciclos `while` y `for`
* Funciones
* Listas y diccionarios
* Validación de entradas
* Números aleatorios
* Lectura y escritura de archivos JSON

## Ejecución

1. Instalar Python 3.
2. Descargar `aventura\\\_reino\\\_oscuro.py`.
3. Abrir una terminal en la carpeta del archivo.
4. Ejecutar:

```bash
python aventura\\\_reino\\\_oscuro.py
```
